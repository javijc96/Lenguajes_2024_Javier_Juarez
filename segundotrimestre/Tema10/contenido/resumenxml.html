Resumen de XML

XML es un estándar para estructurar y compartir datos entre sistemas. Ofrece flexibilidad mediante etiquetas personalizadas y validación mediante DTD o XML Schema.

Definición de XML

Es un lenguaje de marcas cuyo propósito principal es compartir datos a través de diferentes sistemas, como Internet. Es similar a HTML aunque en este las etiquetas se definen de forma manual.

XHML

Es una versión más estricta de HTML que sigue las reglas de XML.

Sigue las reglas de XML para definir la estructura y el contenido de las páginas web. Compatible con HTML y CSS, tiene el mismo uso que HML y su extensión es xhtml.

Aunque ha sido importante en la evolución de los estándares web, en la práctica, HTML5 es la opción utilizada en el desarrollo web actual.

Características de XML

Es un lenguaje de estructura libre no predefinido en el que las etiquetas se deben definir.
  • Consta de dos partes, prólogo y cuerpo.
  • Usa espacios de nombres para evitar ambigüedades.
  • Su sintaxis es como la de HTML.
  • Utiliza esquemas en forma de árbol con una única raíz.
  • Los documentos se validan.

    DTD, Document Tipe Definition

    Metalenguaje que permite la descripción del formato que deben cumplir los datos en un documento XML

    Un DTD describe la estructura del documentos
  • Elementos
  • Orden
    • Ubicación de DTD

  • Mixto

    • Elementos de XML

    Elemento

    Cardinalidad: Número de veces que aparece el elemento descendiente
  • ?: El elemento aparece 0 o 1 vez.
  • *: El elemento aparece 0 a N veces.
  • +: El elemento aparece 1 a N veces.

    • Secuencia: Se indican los elementos que deben de aparecer en el fichero XML.

    X, Y: Indica que el elemento X tiene que aparecer a continuación de Y. (X|Y): Indica que aparece el elemento X o el elemento Y, pero no los dos al mismo tiempo.

    Combinación: Se pueden combinar los símbolos de secuencia y los de cardinalidad
